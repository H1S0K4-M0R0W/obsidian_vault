# - Основы Git

> [!cite] Что такое GitHub?
> `GitHub` - облачная платформа для хостинга IT-проектов, основанная на системе контроля версий
> `Git` - система контроля версий, позволяет:
>  - ==отслеживание изменений в файле==
>  - ==создать и развивать ветви проекта==
>  - ==откат до предыдущей(-их) версии(-й)==
>  - ~*клонирование проекта*~

> [!faq] По-английски
