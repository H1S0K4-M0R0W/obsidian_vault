# - Первоначальная настройка

Стоит начать с того чтобы указать имя и электронную почту пользователя для последующей работы с версиями

``` shell
# Пользовательская эл. почта
git config --global user.email "email.com"

# Пользовательское имя (псевдоним)
git config --global user.name "username"

```
## - Опционально

Можно задать \**красивый (цветной)*\* вывод информации в Консоль с помощью команды:
``` shell
git config --global color.ui "true"

```

Можно задать имя ветви для использования по-умолчанию с помощью команды:
``` shell
git config --global init.defaultbranch "name"

```

Для просмотра всего конфигурационного файла можно воспользоватся данной командой:

``` shell
git config list

# Результат:
user.email=second-name@gmail.com
user.name=AdMiN
init.defaultbranch=main

```

Для просмотра конкретного параметра из конфигурации `Git` можно с помощью команды:
``` shell
# git config парметр
git config --global user.name

# Результат
username
```

> [!warning] Важно
> Если вы пропиши указанные выше команды без параметра `--global` то данные настройки буду применены <u>**ТОЛЬКО**</u> для конкретного репозитория (папки), а не всей системы `Git`

# Расположение конфигурационных файлов `Git`
(Пример для Windows)
 
 1. Системные настройки Git (для всех пользователей)
- C:\Program Files\Git\etc\gitconfig

1. Глобальные настройки (для текущего пользователя)
- %USERPROFILE%\.gitconfig

1. Локальные настройки (для конкретного репозитория)
- <Путь-к-репозиторию>\.git\config
